sushi-sh
======

> Command Line Interface for SushiSwap




# Usage
<!-- usage -->
```sh-session
$ npm install -g sushi-sh
$ sushi COMMAND
running command...
$ sushi (-v|--version|version)
sushi-sh/1.1.0 darwin-x64 node-v14.17.3
$ sushi --help [COMMAND]
USAGE
  $ sushi COMMAND
...
```
